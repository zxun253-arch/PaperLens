# 文献透镜 / PaperLens v0.3.0 发布说明

## 版本定位

v0.3.0 是“研究工作流增强版”。这一版在 v0.2 单篇论文辅助能力基础上，增加文献库管理、跨论文检索、多论文对比和文献综述辅助工作流。

## 新增功能

- 文献标签：支持给论文添加、删除标签，并在论文库中按标签筛选。
- 阅读状态：支持未读、阅读中、已读、归档。
- 重点收藏：支持标记重点论文，并在论文库中只看收藏。
- 论文库筛选：支持关键词、标签、阅读状态、收藏和排序。
- 全局搜索：可搜索论文信息、标签、分块内容、阅读笔记和问答记录。
- 多论文对比：支持选择 2-5 篇论文，生成对比表和多论文对比 Prompt。
- 文献综述辅助：支持生成综述 Prompt，自定义 API 模式下可生成综述草稿，并保存到本地。
- 综述导出：支持将综述草稿导出为 Markdown 和 Word。
- OCR 预留：优化扫描版 PDF 提示，新增 OCR 能力骨架，不默认上传 PDF 到云端。
- 本地模型模式说明增强：补充 Ollama / LM Studio 使用边界和接入建议。
- 导出增强：论文 Markdown / Word 导出加入标签、阅读状态、收藏、AI 结果历史和问答证据。

## 数据库升级

v0.3.0 使用兼容迁移，不会清空旧数据。

- `papers.reading_status`
- `papers.is_favorite`
- `paper_tags`
- `literature_reviews`

## 已知限制

- OCR 仍为预留能力，不是完整 OCR 实现。
- 多论文对比依赖论文已有元信息、分块质量和用户笔记质量。
- 自定义 API 生成的内容仍需结合原文核对。
- 证据定位仍基于 chunk，不等于精确 PDF 页码。
- 本地模型真实效果取决于用户本机模型、显存、服务地址和模型能力。
- Provider 仍需要用户使用真实 API Key、Base URL 和模型名进行手动验收。

## 推荐验收流程

1. 导入并解析多篇论文。
2. 给论文添加标签、阅读状态和重点标记。
3. 使用论文库搜索、标签筛选和收藏筛选。
4. 搜索 chunk、笔记和问答内容，并跳转到详情页。
5. 选择 2-5 篇论文进入多论文对比。
6. 复制多论文对比 Prompt 和综述 Prompt。
7. 在 custom_api 模式下生成对比结果和综述草稿。
8. 保存综述草稿并导出 Markdown / Word。
9. 检查导出文件不包含 API Key、Base URL 或 Provider 配置。
