# 文献透镜 / PaperLens v0.5.0 发布说明

## 版本定位

v0.5.0 是测试验收与产品导出版。该版本不新增业务功能，重点完成自动化质量检查、手动验收准备、真实 Provider 验收准备、敏感信息检查、便携版整理和发布文案准备。

## 已完成功能范围

- 本地 PDF 导入、解析和论文分块
- 本地算法分析、关键词、关键句和论文内容搜索
- Prompt 工作流与外部 AI 回填
- 阅读笔记、AI 输出历史和论文问答
- Markdown / Word 导出
- 文献标签、阅读状态、重点收藏和全局搜索
- 多论文对比和文献综述辅助
- 综述草稿保存和综述 Markdown / Word 导出
- 多 Provider 自定义 API 架构
- Provider 调用诊断日志
- 问答 evidence chunk 定位
- OCR 预留和扫描版 PDF 提示优化
- 本地模型模式说明增强

## 验收范围

v0.5 验收覆盖启动、论文库、PDF 导入解析、标签/状态/收藏、Prompt、笔记、API 配置、论文问答、全局搜索、多论文对比、文献综述、Markdown / Word 导出、打开原 PDF、数据持久化和敏感信息检查。

## Provider 验证情况

当前代码支持：

- OpenAI-compatible 主干
- DeepSeek / Qwen / OpenRouter / Moonshot / 智谱 GLM 等兼容接口
- Anthropic / Claude 独立 adapter
- Gemini / Google AI 独立 adapter

Provider 可用性仍需用户使用自己的 API Key、Base URL 和 Model Name 手动验证。Ollama 当前仍为占位说明，不做真实调用验收。

## 打包说明

Windows 安装包构建依赖 Tauri 所需的 WiX / NSIS 工具链。如果当前环境无法下载相关工具，安装包可能失败；这不代表代码构建失败。可使用 release exe 整理便携版。

## 便携版说明

便携版目录建议为：

```text
release/PaperLens-v0.5.0-portable/
```

用户可双击 `paper-lens.exe` 启动。用户数据默认保存在系统应用数据目录中的本地 SQLite。

## 安全提醒

- 不要提交真实 API Key。
- 不要提交本地数据库、用户 PDF 或导出文件。
- API Key 当前仅保存于本机 SQLite，适合个人本地使用。
- Markdown / Word 导出不包含 API Key、Base URL 或 Provider 配置。
- 诊断日志只保存脱敏摘要。

## 已知限制

- 不支持完整 OCR。
- 本地模型模式仍为预留说明。
- Word 导出是基础格式。
- 问答证据定位基于 chunk，不等于精确页码。
- Provider 需要真实服务商配置手动验收。
